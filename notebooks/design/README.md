### Automated System

![automated-system](./pix/automated-system.jpg)

Why is designing for human-automation (and AI) interaction tricky?
* Dynamic **Humans**
	* Iron of Automation
		* Function (re-)allocation (re-appropriation)
		* Human behaviour changes
	* (Un-)Learning of skill
* Dynamic **Machines**
* Dynamic **Contexts**
* Dynamic **Interaction**
